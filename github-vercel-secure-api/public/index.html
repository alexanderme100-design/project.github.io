✨ 視力保健小幫手

愛眼守則，eye你唷！

  • 用眼3010：近距離用眼每30分鐘，就要休息10分鐘，看看遠方。
  • 正確姿勢：讀書寫字時，腰桿挺直，眼睛與書本保持35-45公分距離。
  • 充足光線：光線要充足且均勻，避免在太亮或太暗的地方看東西。
  • 戶外活動：每天至少2小時的戶外活動，讓眼睛多接觸自然光，可以預防近視喔！
  • 均衡飲食：多吃深綠色蔬菜（如菠菜、花椰菜）和橘黃色蔬果（如胡蘿蔔、南瓜），補充葉黃素。
  • 螢幕設定：使用電子產品時，螢幕亮度要適中，並保持適當距離。
